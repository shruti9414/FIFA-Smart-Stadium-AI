## 💻 LOCAL SETUP (Optional for testing)

### Prerequisites
- Node.js 18+ (already installed)
- MySQL 8.0+ running locally
- Grok API key (if testing Grok endpoints)
- Gemini API key (if testing Gemini endpoints)

### Installation
```bash
cd "C:\Users\Admin\Pictures\FIFA SmartOps AI"

# Install dependencies
npm install

# Setup database
npm run db:init      # Initialize database schema
npm run db:seed      # Seed sample data

# Start dev server
npm run dev          # Runs on http://localhost:3000
```

### Database Setup (if needed)
```bash
# The project expects MySQL on localhost:3306
# Update .env.local with your MySQL credentials

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=fifa_stadium
```
